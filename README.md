
## Installation

```st
Metacello new
	repository: 'github://BenjaVisionPro/CatalystFoundation:main/src';
	baseline: 'CatalystFoundation';
	load
```

## Load Lepiter

After installing with Metacello, you will be able to execute

```
#BaselineOfCatalystFoundation asClass loadLepiter
```
